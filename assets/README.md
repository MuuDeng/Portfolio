# Assets Folder

Place your personal assets in this folder:

## Required Files

- `resume.pdf` - Your resume/CV for download
- `profile.jpg` - Your profile photo (recommended: 400x400px)

## Optional Files

- `project-1.jpg`, `project-2.jpg`, `project-3.jpg` - Screenshots for your projects
- `favicon.png` - Website favicon (32x32px or 16x16px)

## Tips

- Keep images optimized for web (use compression)
- Use JPEG for photos, PNG for graphics with transparency
- Profile image: Square format works best
- Project images: 16:9 or 4:3 aspect ratio recommended
